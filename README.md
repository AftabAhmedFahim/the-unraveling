# The Unraveling

The Unraveling is a Python maze exploration game where players navigate through a labyrinth to reach the goal. The game challenges players to find their way through a dynamically generated maze while avoiding getting lost in the twists and turns.

## Features

* Maze generation and navigation
* Player movement using keyboard controls
* Visual player icon
* Simple and engaging gameplay
* Modular code structure for easy expansion

## Installation

1. Clone the repository

```
git clone https://github.com/AftabAhmedFahim/the-unraveling.git
```

2. Navigate to the project folder

```
cd the-unraveling
```

3. Install dependencies

```
pip install -r requirements.txt
```

## How to Run

Run the game using:

```
python project.py
```

## Controls

- `P`: play (from start screen)
- `E`: exit (from start screen)
- Arrow keys: move player
- `R`: restart current maze
- `M`: open difficulty menu
